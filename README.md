# Семантичний пошук по робочих програмах дисциплін

## Про проект

Дослідницький проект студентського наукового гуртка "Теорія та практика програмної інженерії" кафедри КІТ ННІ ДІІТ УДУНТ.

**Мета:** дослідити ефективність векторних баз даних для семантичного пошуку та аналізу робочих програм дисциплін.
